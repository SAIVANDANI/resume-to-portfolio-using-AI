Portfolio Website Generator A Streamlit-based tool that converts a resume (PDF or DOCX) into a clean, responsive personal portfolio website using HTML, CSS, and JavaScript. The app extracts structured resume content and generates a multi-section static website ready for download.

Features Upload PDF or DOCX resumes Extract and structure content into sections: Header, About, Skills, Experience, Projects, Education, Contact Outputs standalone HTML, CSS, and JavaScript Download website files as a ZIP Simple, developer-focused UI

Tech Stack Python, Streamlit, LangChain, Google Gemini API, PyPDF2, python-docx

How It Works User uploads a resume file (PDF or DOCX) Text is extracted using appropriate parsers Resume content is converted into a structured portfolio layout Static website files (index.html, style.css, script.js) are generated Files are packaged into a ZIP for download

Use Cases Quickly creating a personal portfolio Resume-to-website automation Learning project for Streamlit + LLM integration Demonstrating document processing and content generation
